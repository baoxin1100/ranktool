## ✨ 功能特性
- **自定义栏位**：自由添加、重命名排名等级（如：夯、顶级、NPC、拉完了等）。
- **拖拽排序**：通过简单的拖拽将图片卡片分配至不同等级。
- **尺寸可调**：支持小、中、大三种图片尺寸，且可实时同步更新所有已上传图片。
- **导出截图**：一键将当前的排名结果保存为高清图片。
