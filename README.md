# 鸣潮角色强度排名工具

这是一个用于创建鸣潮角色强度排名的工具，支持自定义排名栏位、拖拽角色卡片以及导出截图。

## 🌐 在线体验
您可以直接访问以下地址使用该工具：
[https://ranktool.baoxin1100.top](https://ranktool.baoxin1100.top)

## ✨ 功能特性
- **自定义栏位**：自由添加、重命名排名等级。
- **拖拽排序**：通过简单的拖拽将角色卡片分配至不同等级。
- **尺寸可调**：支持小、中、大三种图片尺寸，且可实时同步。
- **导出截图**：一键将当前的排名结果保存为高清图片。
